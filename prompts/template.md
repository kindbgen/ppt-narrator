# AI提示词模板

## PPT生成提示词

### 核心任务
根据wiki文档内容生成结构化的PPT页面数据。

### 要求
1. **内容分析**
   - 识别文档的主标题和章节结构
   - 提取每个章节的核心要点
   - 识别关键信息、数据、图表等

2. **页面拆分策略**
   - 主标题页：文档标题 + 简介
   - 章节页：每个一级标题作为独立页面
   - 内容页：二级标题内容适当合并
   - 总结页：核心要点汇总

3. **内容提取原则**
   - 简洁明了：每页不超过5个要点
   - 突出重点：保留关键数据和结论
   - 视觉友好：适合展示的文字量

4. **时长预估**
   - 标题页：30-60秒
   - 内容页：90-180秒
   - 总结页：60-120秒

### 输出格式
```json
{
  "slides": [
    {
      "title": "页面标题",
      "content": "<div>HTML内容</div>",
      "duration": 120,
      "keyPoints": ["要点1", "要点2", "要点3"]
    }
  ]
}
```

### 样式风格说明
- **business**: 商务风格，蓝色主色调，简洁专业
- **tech**: 科技风格，深色背景，现代动感
- **minimal**: 简约风格，浅色调，清爽优雅
- **education**: 教育风格，高对比度，清晰易读

## 旁白生成提示词

### 核心任务
为每页PPT生成演讲者旁白草稿。

### 要求
1. **语言风格**
   - 自然流畅的口语化表达
   - 避免过于书面化的表述
   - 使用通俗易懂的词汇

2. **结构组成**
   - 开场引导：介绍本页主题
   - 内容展开：详细解释要点
   - 过渡衔接：连接下一页或总结

3. **时长控制**
   - 根据页面时长调整内容长度
   - 标题页：简短介绍（30-60秒）
   - 内容页：详细展开（90-180秒）
   - 总结页：回顾总结（60-120秒）

4. **演讲技巧**
   - 添加强调语气（"特别重要的一点是..."）
   - 使用互动语句（"大家可以看到..."）
   - 适当提问（"那么为什么会这样呢？")

5. **关键词提示**
   - 提取3-5个核心关键词
   - 用于演讲时的记忆提示

### 输出格式
```json
{
  "narration": "完整的旁白文本...",
  "keywords": ["关键词1", "关键词2"],
  "tips": ["演讲提示1", "演讲提示2"]
}
```

### 演讲提示示例
- "此处需要放慢语速，强调重点"
- "可以结合手势，指向屏幕上的数据"
- "适当停顿，让听众消化信息"
- "与听众互动，询问是否有疑问"

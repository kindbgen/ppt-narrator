## [1.0.0] - 2025-06-05

### Added
- AI 驱动的 PPT 内容生成，支持主题输入和文档上传
- 双屏演讲模式：演讲者视图 + 观众视图
- 基于 Reveal.js 的演示文稿渲染引擎
- LLM 网关统一接入 (Claude / OpenAI / Ollama)
- Wiki MCP (Docmost) 知识库集成
- SQL.js 本地数据库持久化存储
- Electron 桌面应用支持 (Windows/macOS/Linux)
- NSIS 安装程序 (Windows)
- DMG 磁盘映像 (macOS)
- AppImage + deb 包 (Linux)
- Markdown 转 PPT 幻灯片解析
- PDF 导出功能
- Tailwind CSS 4.x 样式框架
- Vue 3 + Pinia + Vue Router 前端架构
